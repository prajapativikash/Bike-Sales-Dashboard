Project Overview:
This project is an Interactive Bike Sales Dashboard built using Microsoft Excel only.
It includes data cleaning, formatting, pivot tables, pivot charts, slicers, and a final dashboard to analyze bike sales performance.

The dashboard helps in understanding key business insights such as:
Sales by Region
Sales by Gender
Sales by Age Group
Sales by Income Range
Customer Purchase Trends

Tools Used
Microsoft Excel
Pivot Tables
Pivot Charts
Slicers & Timeline
Excel Formulas
Data Cleaning Methods
